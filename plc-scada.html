Introduction
It is impossible to control and supervise on all industrial activities manually. A great leap towards automation, PLC-SCADA is a control architecture used for automation of industrial process and to control remote equipment using ladder logic with microcontrollers. SCADA is an abbreviation for Supervisory Control And Data Acquisition. It is a control system where a computer system monitors and controls a process (like production, manufacturing, refining, etc.). Do not miss this workshop if you are an engineer looking for a job opportunities in the field of automation.
